# 목록(List) 사용법
Unordered 
* Item 1
   * Item 1-1
   * Item 1-2
* Item 2  
  
Ordered 
1. Item 1
   1. Item 1-1
   2. Item 1-2
2. Item 2
3. Item 3

# 문단 구별
1. 한 개 이상의 빈 줄을 문단 사이에 삽입
2. 줄의 마지막에 [Space Bar] 를 두 번 이상 눌러 띄어쓰기

문장을 작성하면 됩니다.(공백을 안 두면..) 
빈 줄이 없으면 자동으로 앞의 문장 뒤에 붙습니다.(Space Bar를 두 번 이상 눌러 띄어쓰기를 하면..)   
위 문장에서 두 칸의 공백을 두어 강제 개행할 수 있습니다.
